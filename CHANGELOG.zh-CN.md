# 更新日志

## 0.2.0 - 2026-07-10

- 移除主界面的计划步骤显示，第二行现在始终只显示活动对话数量。
- 增加旧进程清理和命名互斥体，重复启动后最多保留一个悬浮窗和托盘图标。
- 重做主界面右键菜单，第一次点击即可执行并立即关闭菜单。
- 增加 Windows 任务栏通知区域集成，支持显示、隐藏、设置和退出。
- 启动器和登录启动快捷方式改用 `pythonw.exe`，不再保留控制台窗口。
- 增加透明度、颜色、字体大小、置顶、锁定位置和多显示器 X/Y 坐标持久化。
- 将工具调用计数改为活动对话数量显示。
- 修复隐藏窗口恢复问题，并保留虚拟桌面坐标。
- 修复右键菜单命令分发和 Tk 主线程调度问题。
- 增加中英文 README、文件规范和更新日志。

## 0.1.0 - 2026-07-09

- 初始 Windows 外部伴侣实现。
- 增加本机 Codex app-server 额度轮询和会话活动判断。
- 增加可拖动桌面悬浮窗和基础设置菜单。
