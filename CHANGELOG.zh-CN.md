## 未发布

- 新增事务化设置会话：“应用”只预览运行时值，“保存”才持久化，“关闭”恢复打开设置时的快照。
- 新增统一整数输入校验，支持逐键输入负坐标并拒绝粘贴的非法格式。
- 新增可逆百分比缩放会话，加号和减号对宽高进行对称缩放。
- 新增当前开发路线图及同步的中文翻译。
- 新增校验后的窗口尺寸、缩放模式和 1–10 秒刷新间隔设置。
- 新增弹出菜单工作区定位，以及纯额度日期/最近到期时间格式化 API。
- 新增独立的单实例刷新调度 API 及有限间隔测试。
- 新增六组英文/中文文档副本的结构一致性检查器。
- 新增纯额度健康状态 API，并加入健康/警告/危险悬浮窗颜色。
- 新增可选的空闲收缩和悬停展开，默认保持展开。
- 确保从收缩状态通过设置或托盘恢复时，主界面先恢复完整展开。
- 新增托盘生命周期策略 API，用于动作白名单和单次恢复调度。
- 将自由/等比例窗口缩放抽为独立并经过测试的 API。
- 新增中英文兼容性矩阵，并明确实体测试发布门槛。
- 新增仅限本地的额度数据源规范化 API，并明确禁止令牌边界。
- 新增可重复的自动化发布门禁运行器，并将实体测试保持为独立门槛。
- 在中英文开发计划和 API 规范中加入 Git/GitHub 变更纪律。
- 修复 Windows 编辑器和 PowerShell 常见 UTF-8 BOM 设置文件的读取问题。
- 新增右下角菜单定位、副屏幕和额度格式化回归测试。

## 0.2.0 - 2026-07-10

- 增加配置、活动状态、运行时、诊断和测试边界 API，并记录重大行为及性能变更规范。
- 将单实例启动改为命名互斥体，并增加无控制台模式下的诊断日志。
- 加强配置校验、活动状态超时和 PowerShell 启动器检查。
- 用可测试的 Tk 弹出适配层替代原先首次点击不稳定的原生菜单路径。
- 移除主界面的计划步骤显示，第二行现在始终只显示活动对话数量。
- 增加旧进程清理和命名互斥体，重复启动后最多保留一个悬浮窗和托盘图标。
- 重做主界面右键菜单，第一次点击即可执行并立即关闭菜单。
- 增加 Windows 任务栏通知区域集成，支持显示、隐藏、设置和退出。
- 启动器和登录启动快捷方式改用 `pythonw.exe`，不再保留控制台窗口。
- 增加透明度、颜色、字体大小、置顶、锁定位置和多显示器 X/Y 坐标持久化。
- 将工具调用计数改为活动对话数量显示。
- 修复隐藏窗口恢复问题，并保留虚拟桌面坐标。
- 修复右键菜单命令分发和 Tk 主线程调度问题。
- 增加中英文 README、文件规范和更新日志。
