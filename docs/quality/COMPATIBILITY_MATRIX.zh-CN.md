## 发布门槛

Windows 11 x64 是受支持平台。在每个具有阻塞性的“待验证”或“部分完成”项目获得实体 Windows 结果，或由维护者明确批准并记录环境限制之前，不得标记产品已达到发布就绪状态。明确标记为“非阻塞”的项目仍会报告，但不进入可执行阻塞项集合。
