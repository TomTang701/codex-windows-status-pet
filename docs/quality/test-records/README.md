# Physical Test Records

This directory stores detailed Windows evidence behind the summary in [`../COMPATIBILITY_MATRIX.md`](../COMPATIBILITY_MATRIX.md).

Each record must state the date, commit, application version, Windows build, monitor topology, DPI, taskbar position, result, limitations, steps, expected behavior, actual behavior, and safe evidence. A simulated or headless result must not be recorded as a physical pass.

Raw probe JSON is machine-readable evidence. It may be English-only when a translated copy would add no verification value and must not contain credentials, session content, or project files.
