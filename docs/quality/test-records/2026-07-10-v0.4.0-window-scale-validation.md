## Method

The current-build `Pet` and real Windows Tk font/widget stack were instantiated with dummy transport and tray adapters so no Codex quota request, activity scan, external service, or user configuration write was needed. A temporary schema-1 settings path was used for persistence checks.

At each scale the test inspected actual Tk geometry, `Segoe UI` and `Segoe UI Emoji` sizes, wraplength, pack padding, five stable row widgets, Reset Credit date text, bottom-right secondary-monitor menu placement, drag/lock behavior, Hide/Show, Compact/Expand, atomic Save, and reload.

The running branch application was then shut down through its Tk `WM_DELETE_WINDOW` protocol and restarted from the repository root. Process, launcher, HWND, topology, and CommandLine were inspected after stabilization.
