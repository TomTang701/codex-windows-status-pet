# v0.6.2 Quota Row Visibility and Dynamic Distribution Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let users independently hide the five-hour, weekly, and reset-credit text rows while each remaining row shares the existing text region equally and every quota value remains truthful.

**Architecture:** `config_api` is the single persistence/default boundary; `SettingsSession` keeps its existing transactional contract. `StatusRows` owns five persistent Tk labels and changes only their geometry participation. `Pet.apply_settings()` supplies the normalized settings to that owner; neither the main window nor compact mode recalculates quota or battery presentation.

**Tech Stack:** Python 3, Tkinter, unittest, existing Windows DPI and package/RC scripts.

## Global Constraints

- Use exactly five stable row IDs: `activity`, `progress`, `primary_5h`, `weekly`, `reset_credit`; never add a sixth row.
- New booleans are `show_primary_5h`, `show_weekly`, `show_reset_credit`, default `True`; activity and progress are always visible.
- A hidden row must be unmanaged, not blank; an enabled unavailable/stale row keeps its rendered text.
- Keep official app-server quota authority, classified weekly default battery source, 10-cell battery geometry, WindowMetrics, compact battery-only semantics, Shell identity, and mixed-DPI recovery unchanged.
- Do not add dependencies, generic layout engines, manual row coordinates, font resizing, or a second battery/quota semantics authority.
- Preserve the existing Apply/Save/Close/Restore Defaults transaction semantics.

def _visible_ids(self):
    return tuple(row_id for row_id, label in self.rows.labels.items() if label.winfo_ismapped())

def _centers_x2(self):
    return tuple(2 * label.winfo_y() + label.winfo_height() for label in self.rows.labels.values() if label.winfo_ismapped())

def test_hidden_weekly_reflows_remaining_rows_evenly(self):
    self.rows.pack(fill="both", expand=True)
    self.root.deiconify()
    self.root.geometry("300x200")
    self.rows.set_visible_rows({"show_weekly": False})
    self.root.update_idletasks()
    centers = self._centers_x2()
    self.assertEqual(self._visible_ids(), ("activity", "progress", "primary_5h", "reset_credit"))
    self.assertEqual(len(set(b - a for a, b in zip(centers, centers[1:]))), 1)
```

def set_visible_rows(self, settings):
    visible = {
        "activity": True,
        "progress": True,
        "primary_5h": bool(settings.get("show_primary_5h", True)),
        "weekly": bool(settings.get("show_weekly", True)),
        "reset_credit": bool(settings.get("show_reset_credit", True)),
    }
    for label in self.labels.values():
        label.pack_forget()
    for row_id in ROW_IDS:
        if visible[row_id]:
            self.labels[row_id].pack(fill="x", expand=True, anchor="w")
```
